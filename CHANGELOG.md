# Changelog

## v0.1.0 - 2026-04-10

- Canonical memory engine with SQLite store
- SPO claim extraction with LLM structured outputs
- Deterministic reconciliation with typed conflicts
- Bitemporal semantics using system time and valid time
- Confidence model with component-level clamping
- Five-mode retrieval strategy with fallback chain
- Provenance-aware graph traversal for source tracing
- Memory consolidation, decay, and episodic distillation
- File mirrors for `working`, `knowledge`, and `sources`
- Qdrant and Grafeo projection adapters with stable interfaces
- Golden-scenario eval harness with a naive-RAG baseline
- Local Ollama support verified with `qwen3.5:latest`
